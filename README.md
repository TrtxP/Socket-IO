# Simple chat for discussing, based on Node.JS, Express.JS, and Socket.IO technologies

---

## Features

### 1. Adding chat rooms
### 2. Switching the chats
### 3. Log in and Sign up forms 
###    for registration and autho-
###    rization users in the chat
###    server
### 4. All user data is saved
###    in database PostgreSQL
### 5. Hashing user passwords
###    via using bcrypt module
---
## Project launching

---
### 1. Clone project
```
git clone https://github.com/TrtxP/Socket-IO
```
---
### 2. Initialize modules Node.JS
```
npm i
```
---
### 3. Launch the project
```
npm start
```
